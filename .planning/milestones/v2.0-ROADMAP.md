## Notable Load-Bearing Decisions

- **D-04 two/three-commit bisect shape** — each dep swap commits separately from content changes so `git bisect` isolates dep regressions from content regressions. Extended across Phases 9-13 with phase-specific variants (D-09-10 three-commit, D-10-10 two-commit, D-11-09 two-commit, D-12-06 per-directory batches).

- **Minimum-scope derogation** — used across Phase 8 (legacy-peer-deps bridge), Phase 9 (tree-kill + chalk scope expansion), Phase 12 (main-utils hoisted-require workaround). All derogations documented in SUMMARY files with retirement targets.

- **`bot/**` scope isolation** — pre-existing dirty state in `bot/src/**` (21 files) was never staged or committed across any v2.0 phase. Bot tests still 294/294 unchanged (Phase 10 verified baseline).

- **`electron-vite` as Vite wrapper** — chose opinionated tool over hand-rolled `vite-plugin-electron`. Subsumed the custom `_scripts/dev-runner.js` orchestrator (retired 369 LOC).

- **Vue 3 Options API preserved** — Phase 8 kept Options API throughout, matching v1.4's decision. Composition API only where Vue 3 strictly requires it.
