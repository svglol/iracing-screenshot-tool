- **`export function X()` over `module.exports = {X}` in TS** (v2.0 Phase 12 Plan 01) ‚Äî Using `module.exports = {...}` without any `export` or `import` keyword makes a `.ts` file a SCRIPT (global scope), not a MODULE. tsc reports `TS6200 Definitions conflict` / `TS2451 Cannot redeclare` cross-file when two scripts declare same-named identifiers.
